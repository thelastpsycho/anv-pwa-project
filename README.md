1. Clone the repository

```bash
git clone https://github.com/yourusername/anvaya-bali.git
cd anvaya-bali
```

2. Install dependencies

```bash
npm install

```

3. Start development server

```bash
npm run dev
# or

```

4. Build for production

```bash
npm run build
# or

```
